# ❌ Tic Tac Toe - Pro Edition ⭕

A modern, responsive take on the classic Tic Tac Toe game. Built with pure **JavaScript**, this project focuses on sleek UI/UX, smooth transitions, and logical win-state detection.

---

## ✨ Pro Features
- **Dynamic Win Highlighting:** The winning combination glows green to clearly signal the end of the match.
- **Visual Distinction:** 'X' and 'O' are color-coded (Blue vs. Red) for better readability.
- **Smart Logic:** Prevents double-clicking or overwriting occupied cells and handles draw states gracefully.
- **Dark Mode UI:** Designed with a modern, high-contrast dark theme for a premium feel.
- **Responsive Board:** Built with CSS Grid to ensure a perfect layout across different screen sizes.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Transitions & Grid)
- **Logic:**  JavaScript (DOM Manipulation & ES6)

## 🎮 How to Play
1. The game starts with **Player X**.
2. Click on any empty square to place your mark.
3. Align three marks horizontally, vertically, or diagonally to win.
4. Hit the **Reset Game** button to start a fresh match at any time.
